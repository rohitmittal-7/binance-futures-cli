# Binance Futures Testnet CLI App

## 📌 Project Overview
This is a Python CLI-based trading application that connects to Binance Futures Testnet (USDT-M) and allows users to place MARKET and LIMIT orders.

---

## ⚙️ Features
- Place MARKET orders
- Place LIMIT orders
- Supports BUY and SELL positions
- Interactive CLI input
- Logging of API requests and responses
- Error handling for API and input issues

---

## 🛠️ Tech Stack
- Python 3
- Requests library
- Binance Futures Testnet API
- CLI (Argparse / Input-based)

---
